# Memory---Game---Java :
Memory Game is a simple console-based Java game where players match pairs of hidden cards. The game shuffles cards randomly and challenges players to remember card positions to find all matching pairs.

Features : 
- Random card shuffling
- Interactive console gameplay
- Input validation for card selection
- Match and mismatch detection
- Win message upon finding all pairs
  
Technologies Used :
- Java
- Arrays
- Random Class
- Scanner Class
